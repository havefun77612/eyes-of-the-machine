# Eyes of the Machine: The Zero-Math Guide to AI & Computer Vision

**Author:** A.L Hossam A. Abdelwahab

A 10-chapter educational book designed for **college freshmen** — no math background, no coding background. Take the reader from "what is a pixel?" to building a real AI Computer Vision project in Python.

## Target Audience

- College freshmen (Level 1)
- Zero math background
- Zero coding background

## Structure

| Chapter | Title | Topic |
|---|---|---|
| 1 | The Silicon Brain vs. The Human Mind | Philosophy of AI |
| 2 | The Workshop | Colab vs. Local setup |
| 3 | The Universal Language | Python basics |
| 4 | The Anatomy of a Snap | Pixels, RGB, images as numbers |
| 5 | The Digital Canvas | Image processing (OpenCV/PIL) |
| 6 | Finding the Edges | Feature extraction / edge detection |
| 7 | The Data Universe | Datasets & Kaggle |
| 8 | The Awakening | Intro to Machine Learning |
| 9 | The Masterpiece | Face Detection project |
| 10 | The Human Symbiosis | Ethics & future |

## Quick Start (for readers)

Open in Google Colab:

1. Go to [Google Colab](https://colab.research.google.com/)
2. File → Open Notebook → GitHub → paste this repo URL
3. Start with `notebooks/03_python_basics.ipynb`

Or read the chapters directly in `chapters/` folder.
